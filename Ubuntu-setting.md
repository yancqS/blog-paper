# Ubuntu 18.04LTS系统设置窗口打不开或者消失解决办法

## 系统设置打不开，请重新安装gnome-control-center

命令：`sudo apt-get install gnome-control-center`

## 系統设置里只有很少的几个图标请重新安装unity-control-center

命令：`sudo apt-get install unity-control-center`
